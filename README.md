**Steps to run MICROKART (e-commerce website)**

1.clone the frontend as well as backend.

2.for frontend:
  -npm install,
  -npm start

3.for backend
  -npm install,
  -npm start
  

4.now register , then login .

5.you will see e-commerce website.

6.enjoy!
